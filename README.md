# Yappening

## Coded in Python

### A platform for AI VS AI and AI VS Human Debating on any topic

#### Features:
1. Variable Number of Rounds
2. AI Judgement at end
3. AI Scoring of Each Argument
4. Different AI Personalities
5. Efficient Error Handling

#### How to test it:

1. Go over to [Yappening](https://yappening-lsgbmrj6pqfke69mmn8sjr.streamlit.app/)
2. Test it :)

#### How to run it locally?

1. Clone the repository.
2. Create a .env file with your Hack Club AI and Gemini API Keys
3. Run `pip install -r requirements.txt`
4. Run main.py file using `python3 main.py`
